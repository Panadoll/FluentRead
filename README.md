# 流畅阅读 (FluentRead)

精简版本

## 🚀 快速使用

- **自动翻译**：安装并启用后，进入非中文网站会自动开始全文双语翻译。
- **撤销翻译**：右键页面 → `FluentRead` → `撤销翻译`。
- **加入黑名单并撤销翻译**：右键页面 → `FluentRead` → `加入黑名单并撤销翻译`（以后该站点不再自动翻译）。
- **从黑名单移除**：右键页面 → `FluentRead` → `从黑名单移除`，或在 Popup 面板的 `站点黑名单` 中移除。
- **清除翻译缓存**：打开 Popup 面板 → 底部点击 `清除翻译缓存`（用于重置当前页面缓存并重新翻译）。
