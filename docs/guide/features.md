# 功能介绍

流畅阅读已精简为“**非中文网站自动全文翻译为中文**”的最小体验。

## 核心功能

### 自动全文翻译（零操作）
进入非中文网站后，自动开始全文双语翻译，无需任何点击或快捷键。

### 双语对照显示
仅保留双语对照模式，并固定为“朴素样式”，阅读干扰最小。

### 站点黑名单（主域名）
当某个站点不需要自动翻译时，可快速加入黑名单，之后不再自动翻译。

## 翻译引擎
只保留 3 个可选引擎：

- 微软翻译
- 谷歌翻译
- 本地模型 (Ollama)

## 常见问题

### 翻译没有反应？
1. 检查插件是否正确安装并启用
2. 确认当前站点未被加入黑名单
3. 尝试刷新页面或重启浏览器

### 翻译结果不准确？
1. 尝试切换不同翻译引擎
2. 使用本地模型 (Ollama) 并选择质量更高的模型
